# рҹҺҜ AI Skill Assessment & Learning Plan Agent

An AI agent that goes beyond resume screening вҖ” it **conversationally tests real skill proficiency**, 
identifies gaps against a job description, and generates a **personalised learning roadmap** 
with curated resources and time estimates.

---

## рҹ’Ў The Problem

Resumes show **claims**, not **proof**. Listing "Kubernetes" doesn't mean production-level expertise.
This agent finds the truth through adaptive questioning and builds a plan to close the gaps.

---

## рҹ”§ Approach

1. **Extract** вҖ” AI parses JD and resume, identifies required skills, overlaps, and gaps
2. **Assess** вҖ” Adaptive chat-based Q&A per skill (difficulty adjusts based on answers)
3. **Score** вҖ” Each response scored 0-5 with weighted averaging
4. **Analyze** вҖ” Gap analysis with priority ranking: `gap Г— weight Г— category_multiplier`
5. **Plan** вҖ” Personalised learning path leveraging adjacent skill transferability
